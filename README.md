# Streamlit ML App - Despliegue con Docker

Este proyecto contiene aplicaciones de Machine Learning desarrolladas con Streamlit, listas para ser desplegadas usando Docker.

## Requisitos Previos

- Git instalado
- Docker Desktop instalado
- Cuenta en GitHub
- Cuenta en DockerHub
- Python 3.12+ (opcional, para desarrollo local)

## Crear tu Repositorio desde este Template

### 1. Clonar el Repositorio Original

```bash
git clone https://github.com/Factoria-F5-madrid/DA-Workshop-Streamlit-deployment.git
```
```bash
cd DA-Workshop-Streamlit-deployment
```

### 2. Crear tu Propio Repositorio

1. **Ve a GitHub.com** y crea un nuevo repositorio vac√≠o
2. **NO inicialices** con README, .gitignore o licencia
3. **Copia la URL** de tu nuevo repositorio

### 3. Cambiar el Origen Remoto

Eliminar el origen actual
```bash
git remote remove origin
```

Agregar tu nuevo repositorio como origen
```bash
git remote add origin https://github.com/TU-USUARIO/TU-NUEVO-REPO.git
```

Verificar el cambio
```bash
git remote -v
```

### 4. Subir a tu Repositorio

Hacer commit de cualquier cambio local
```bash
git add .
```
```bash
git commit -m "Initial commit: Streamlit ML apps with Docker"
```
Subir a tu repositorio
```bash
git push -u origin main
```
